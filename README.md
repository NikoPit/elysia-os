## Building and Running

### Prerequisites
- Rust nightly toolchain
- `bootimage` crate for creating bootable images
- QEMU for emulation

### Building
```bash
# Install bootimage
cargo install bootimage

# Build the OS
cargo build

# Build and run in QEMU
cargo run
```

### Testing
```bash
# Run all tests
cargo test

# Run specific test suites
cargo test --test stack_overflow
cargo test --test multitask
cargo test --test filesystem
```
